# apollo-backend
## 项目概述

Apollo Backend 是一个使用 Go 语言开发的灰度发布系统后端。项目专注于 Go 技术栈，提供灰度发布系统对应的网络服务

✨ 主要特性

* 纯 Go 技术栈：不依赖重型框架，采用 Go 标准库及轻量级第三方库，代码清晰，便于学习与定制。

* 模块化设计：代码结构清晰，遵循良好的软件设计原则，便于功能扩展和维护。

* RESTful API：提供标准的 RESTful 接口，易于与任何前端（如 Vue.js, React）集成。


🛠 技术栈

语言: Go 1.24.7

Web 框架: Gin

数据库: MongoDB

ORM: github.com/go.mongodb.org/mongo-driver
